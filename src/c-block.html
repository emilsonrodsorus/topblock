What is a rich text element?

Instead of having to add and format headlines, paragraphs, blockquotes, photos, and video separately, the rich text element lets you to generate and format them all in one spot. Simply double-click to generate content.

Editing static and dynamic content

With static or dynamic content, a rich text element can be employed. Simply drop static material into any page and start modifying. Add a rich text field to any collection and then connect a rich text element to that field in the settings panel for dynamic content. Voila!

How to make each rich text's formatting unique

After a class is assigned to the rich text element, the "When within of" nested selector system can be used to style headings, paragraphs, blockquotes, figures, images, and figure captions.
